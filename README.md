# JEE Project

## Environment
+ Intellij 2020.2.3
+ Maven
+ Java 15
+ Tomcat 9.0.41

## Launch project
+ Launch Intellij
+ Add Project Version Control
+ Add Configuration ... :
    + First : Select Tomcat Server
    + Then : Add Tomcat Server file, Select Navigator and Add Artifact war
+ Build and Run
